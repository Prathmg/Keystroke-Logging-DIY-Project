# Keystroke-Logging-DIY-Project
A Python Tkinter-based GUI application that captures and logs keyboard input within the app window, storing data in JSON and text formats to demonstrate event-driven programming and file handling.
📘 README.md
📌 Keyboard Activity Logger (GUI-Based)
📝 Overview

The Keyboard Activity Logger is a beginner-friendly Python project that demonstrates how keyboard inputs can be captured, processed, and stored using a graphical user interface. Built with Tkinter, the application records key events within its active window and saves them in both JSON and text file formats for analysis and learning purposes.

🎯 Features
📊 Real-time keyboard input display
🧾 Logs key events (Pressed / Released)
💾 Saves data in:
JSON format (structured data)
Text file (readable format)
🖥️ Simple and interactive GUI
🔐 Works only within the application window (safe & ethical)
🛠️ Technologies Used
Python
Tkinter (GUI)
JSON
File Handling
🚀 How It Works
Launch the application
Click the Start button
Press keys inside the application window
View real-time updates on the screen
Logs are automatically saved in:
logs.json
logs.txt

🎓 Learning Outcomes
Understanding event-driven programming
Working with GUI in Python
Handling keyboard inputs
Reading and writing JSON and text files
🔐 Ethical Note

This project is designed for educational purposes only.
It captures keyboard input only within the application window and does not monitor system-wide activity.
